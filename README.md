# tmux-config

## Install

```sh
git clone https://github.com/kaar/tmux-config $HOME/.config/tmux
git clone https://github.com/tmux-plugins/tpm $HOME/.config/tmux/plugins/tpm
# Then: prefix + I to install plugins
```
